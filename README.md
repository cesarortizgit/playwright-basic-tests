# Playwright Basic Tests

This is my first Playwright automation project using TypeScript.

## Objective

Automate 3 basic test cases over a demo web application:

1. Open a demo page.
2. Validate the page title.
3. Validate visible text.
4. Click/type into an input and create a new item.

## Tech Stack

- Playwright
- TypeScript
- Node.js
- VS Code

## What I Learned

- How to create a Playwright project.
- How to write basic automated tests.
- How to use locators.
- How to use assertions.
- How to execute tests in headless and headed mode.
- How to open the Playwright HTML report.
